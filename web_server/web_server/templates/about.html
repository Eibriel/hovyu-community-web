Acerca de Hovyũ

Hovyũ (suena similar a hovujú) es una comunidad de comercios e instituciones relacionados Diseño Sustentable, el Comercio Justo y la Alimentación Consciente. La meta de Hovyũ es ser la base de datos universal mas completa alimentada y mantenida por la comunidad.
Está desarrollada por Eibriel como un proyecto de software libre, y no busca ser una competencia frente a otras funtes de datos Sustentables (como Greendy), sino un complemento.
La información es abierta, bajo licencia CC-BY 4.0, y es accesible a través de APIs premitiendo la creación de aplicaciones por parte de terceros.

El compromiso

Los servidores están ubicados en Santa Fe y Buenos Aires (Argentina) apostando a la infraestructura local, utilizamos medios electrónicos para reducir la utilización de papel.
Hovyũ alienta la utilización de soluciones de pago alternativas y descentralizadas, el software libre y las base de datos libres.
